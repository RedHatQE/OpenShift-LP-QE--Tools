# Prow Analyzer AI Initialization

## Environment Requirements

### Required Environment Variables

**For CLI:**
```bash
export SHIP_HELP_MCP_URL="https://ship-help-mcp-continuous-release-tooling--ship-help-bot.apps.gpc.ocp-hub.prod.psi.redhat.com/personas/ocp_ai_helpdesk/mcp"
export SHIP_HELP_MCP_TOKEN="<your-token>"
```

**For Slack Bot (additional):**
```bash
export SLACK_BOT_TOKEN="xoxb-..."
export SLACK_APP_TOKEN="xapp-..."
export MONITORED_CHANNELS="C12345678,C87654321"
export PROMPT_TEMPLATE="<optional-custom-prompt>"
```

### Obtaining Credentials

- **SHIP_HELP_MCP_TOKEN**: Get from #ship-users in Slack or reuse existing token from ship-help-bot
- **Slack tokens**: Create app at https://api.slack.com/apps
  - Enable Socket Mode
  - Required scopes: `channels:history`, `chat:write`
  - Get Bot Token (xoxb-...) and App Token (xapp-...)

## Build Instructions

### CLI Binary
```bash
cd apps/prow-analyzer
go build ./cmd/prow-analyzer--cli
```

### Slack Bot Binary
```bash
cd apps/prow-analyzer
go build ./cmd/prow-analyzer--bot
```

### Container Image
```bash
# From repository root
make -C image/container/prow-analyzer build IMAGE_TAG=v1.0.0
make -C image/container/prow-analyzer push IMAGE_TAG=v1.0.0
```

## Running

### CLI Usage
```bash
./prow-analyzer--cli analyze <prow-url>
```

### Slack Bot (Local)
```bash
./prow-analyzer--bot
```

### OpenShift Deployment
```bash
# Create secrets
oc create secret generic prow-analyzer-secrets \
  --from-literal=ship-help-token="$SHIP_HELP_MCP_TOKEN" \
  --from-literal=slack-bot-token="$SLACK_BOT_TOKEN" \
  --from-literal=slack-app-token="$SLACK_APP_TOKEN" \
  -n prow-analyzer

# Deploy
oc apply -f apps/prow-analyzer/deploy/openshift/deployment.yaml
```

## Testing

## Architecture

- **CLI** (`cmd/prow-analyzer--cli`): Single job analysis
- **Bot** (`cmd/prow-analyzer--bot`): Slack monitoring with automatic analysis
- **Analyzer** (`pkg/analyzer`): Core MCP integration logic
- **Handler** (`pkg/slack/handler`): Slack event processing

## Dependencies

- Go 1.22+
- Ship-help MCP access
- (Optional) Slack app credentials for bot functionality
